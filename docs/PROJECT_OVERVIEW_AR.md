## 1. ما هي أكاديمية كمبيوتك؟

أكاديمية كمبيوتك هي **منصة تعليمية عبر الإنترنت وحضورياً** مصممة للسوق العراقي. يتعلم الطلاب المهارات من خلال الدورات والبرامج، ويحصلون على شهادات، ويثبتون قدراتهم، ويتواصلون مع أصحاب العمل — في مكان واحد.

### الرؤية

| الركيزة | ماذا تعني |
|--------|--------------|
| 🎓 **التعلم** | يأخذ الطلاب دورات وبرامج (عبر الإنترنت أو حضورياً) |
| 🏆 **الإثبات** | يحصل الطلاب على شهادات ومهارات موثقة |
| 💼 **التواصل** | يبحث مسؤولو التوظيف في مجمع المواهب لتوظيف الخريجين |
| 🏛️ **الشراكة** | تتابع الجامعات مسار طلابها على المنصة |

---

## 2. لمن هذه المنصة؟

تخدم المنصة **أربعة أنواع من المستخدمين**:

### 🎓 الطالب (الدور الافتراضي)
المستخدم الرئيسي. يبدأ كل حساب كطالب. يتصفح الطلاب الدورات، يشتركون، يتعلمون، يكملون الاختبارات وتحديات البرمجة، يحصلون على الشهادات، ويبنون ملفاً شخصياً احترافياً.

### 💼 مسؤول الموارد البشرية (HR)
يمثل شركة أو مدير توظيف. يقوم مسؤولو الموارد البشرية بنشر إعلانات الوظائف، والبحث في مجمع المواهب، وعرض ملفات المرشحين، والاشتراك في باقات التوظيف.

### 🏛️ المؤسسة
تمثل جامعة أو معهد. تقوم المؤسسات بالتسجيل، وتتبع تقدم طلابها، ومعرفة ترتيبها بين المؤسسات الأخرى.

### 🛡️ المشرف (Admin)
فريق الأكاديمية. يدير المشرفون جميع المحتويات والمستخدمين والمدفوعات والإعدادات من خلال لوحة تحكم الإدارة.

## 5. الدورات والبرامج

### ما الفرق؟

| | الدورة (Course) | البرنامج (Program) |
|-|--------|---------|
| **الوصف** | موضوع واحد | مجموعة دورات لهدف واحد |
| **مثال** | "أساسيات JavaScript" | "مطور ويب متكامل" (6 دورات) |

### الإلكتروني ضد الحضوري

| الجانب | الإلكتروني (Online) | الحضوري (On-Site) |
|--------|--------|---------|
| **التوصيل** | دروس فيديو من أي مكان | حضور شخصي في موقع |
| **الدفع** | زين كاش (إلكتروني) | نقداً فقط |
| **الاشتراك** | فوري بعد الدفع | تسجيل ← موافقة الإدارة ← الحضور |
| **الميزات** | مشغل الدورة، اختبارات، برمجة | تتبع الحضور، القبول |

### صفحة الدورة تحتوي على:
العنوان، ملخص، ماذا ستتعلم، المتطلبات المسبقة، المدة الزمنية، قائمة الدروس، الاختبارات المشمولة، توفر الشهادة، السعر، زر الشراء، شارة إلكتروني/حضوري، حالة مفتوح/مغلق، التعليقات والتقييمات.

### صفحة البرنامج تحتوي على:
العنوان، ملخص، الهدف من البرنامج، الجمهور المستهدف، المتطلبات المسبقة، قائمة كاملة بالدورات المشمولة، معلومات الشريك، المدة الزمنية، توفر الشهادة، السعر، زر الشراء، شارة إلكتروني/حضوري، التقييمات.

## 6. تجربة الطالب

### تُظهر لوحة التحكم:
- 📚 دوراتي مع أشرطة التقدم
- 🎯 برامجي مع التقدم
- ⏰ المواعيد النهائية القادمة
- 🏅 الشهادات
- 🔔 الإشعارات

### تجربة التعلم

- **مشغل الفيديو** — مشاهدة الدروس مع تتبع التقدم
- **الاختبارات المدمجة** — اختيارات متعددة وصح/خطأ أثناء أو بعد الفيديوهات
- **تحديات البرمجة** — كتابة وتشغيل الكود ضد حالات الاختبار (Test cases)
- **الحضور** — للدورات الحضورية: حضر، غاب، لم يُعقد بعد

### الملف الشخصي للطالب (جاهز كبديل للينكد إن)

| القسم | ماذا يتضمن |
|---------|----------------|
| **المعلومات الأساسية** | الاسم، الصورة، العنوان الوظيفي، النبذة، المدينة، الهاتف، البريد |
| **التعليم** | الجامعة، القسم، الدرجة، السنوات، الحالة |
| **المهارات** | يدوية + موثقة من الدورات/البرامج/الامتحانات المستقلة |
| **التعلم** | الدورات والبرامج المسجل بها والمنتهية |
| **النقاط** | إجمالي النقاط (إلكتروني + حضوري مجموعين) |
| **المشاريع** | العنوان، الوصف، الروابط، جيت هب |
| **المسار المهني** | السيرة الذاتية (CV)، الدور المفضل/المدينة المفضلة، التفرغ |
| **الروابط الاجتماعية** | لينكد إن، جيت هب، موقع الأعمال (بورتفوليو) |

### الامتحانات والشهادات
- **الامتحانات المستقلة** — اجتياز امتحان بدون دراسة الدورة لإثبات المهارة
- **الشهادات** — شهادة PDF قابلة للتحميل برقم فريد للدورات والبرامج والامتحانات المستقلة.

### ميزات التفاعل والمشاركة
- **بوت التوجيه الذكي (AI Chatbot)** — يساعد الطالب على معرفة ما يجب أن يتعلمه بناءً على الهدف والميزانية
- **المجتمع** — طبقة مجتمع بسيطة لتفاعل الطلاب مع بعضهم حول الدورات
- **ميزات الاحتفاظ** — إشعارات وتنبيهات للمواعيد النهائية، والتقدم
- **التعليقات والتقييمات** — يُسمح بالتقييم 1-5 نجوم أو التعليق فقط لمن أتم عملية الشراء
- **دعم وتمييز الطلاب** — ميزات لتمييز (طالب الشهر/طالب الدورة) كمرحلة لاحقة للتفاعل.

## 7. تجربة الموارد البشرية (HR)

### ما يمكن لمستخدمي الموارد البشرية فعله:

1. **نشر وظائف** — إرسال إعلانات توظيف يراها الطلاب
2. **البحث في المواهب** — بحث وتصفية بالمهارات، الجامعة، المدينة، النقاط
3. **عرض المرشحين** — فتح الملفات الشخصية الكاملة للطلاب
4. **إدارة الباقة** — الاشتراك في الباقات للتحكم بالصلاحيات

### ملف الشركة يحتوي على:
معلومات الشركة (الاسم، الشعار، المجال، الحجم)، معلومات الاتصال، تفضيلات التوظيف، واستخدام الباقة.

## 10. الدفع والإيرادات

### مصادر الإيرادات

| المصدر | نموذج الدفع |
|--------|-------|
| **البرامج والدورات الإلكترونية** | شراء لمرة واحدة ← وصول فوري للمعرفة |
| **باقات الشركات HR** | اشتراكات شهرية متجددة |
| **الدورات الحضورية** | الدفع نقداً عند الوصول لمقر التدريب |

### مسار الدفع الإلكتروني

1. ينقر الطالب على **شراء** ← صفحة الدفع
2. يدفع باستخدام **زين كاش (ZainCash)**
3. يتم تأكيد الدفع ← **تحصيل الاشتراك الفوري**

### مسار الدفع الحضوري

1. يقوم الطالب **بالتسجيل** للدورة الحضورية
2. يقوم المشرف **بتأكيد/قبول** التسجيل
3. يدفع الطالب **نقداً** في المقر
4. يقوم المشرف بتحديث حالة الدفع ليصبح التسجيل نشطاً بالكامل

| لوحات المدربين ومهام صلاحية المحاضر | الإدارة (Admin) تتولى الرفع والنشر للمواد |
| نظام ألعاب التعليم والنقاط (شامل أوسمة وتنافس) | الإثبات بالشهادات، وسيتم تحسينه بالإصدارين القادمين |
| خطط التعلم والمسارات الكبرى (Learning Paths) | البحث في نفس القسم أو القسم الفرعي |
| جلسات المتابعة والتوجيه (Mentorship Sessions) | لايوجد (غير متوفر) |
| نظام الكوبونات والخصومات التسويقي | تقديم التسعير كعروض والاعتماد على الخصم المباشر من الأدمن |
| أنظمة الاسترداد المالي الذكي (Refund Policy) | يتم الرد عن طريق تقديم الإلغاء بشكل يدوي وحل المشاكل بالواتساب |
| حماية المحتوى والشهادات ونظام الحقوق (DRM) | الحماية البسيطة الخاصة بالعنوان السري المباشر. |
| نظام الانتظار وقوائم الرغبة لدخول المجموعات (Waitlists) | الترتيب والحجز بالتواصل اليدوي |
| تعليقات وتقييمات خریجی النظام السابق أو بالصور (Testimonials) | التقييمات اليدوية من إدارة النظام عبر موقع الـLanding. |
| خطة قبول ودفع منح وخصومات المؤسسات الكبرى | إضافة أعضاء المنظمة مجاناً وتدقيق الدفع بشكل كلاسيكي أو ورقي |
| تحليلات ومؤشرات النظام والإحصاءات الداخلية (Analytics) | إحصائية الفيلامنت الافتراضية للطلاب. |
| ميزة الأمان المصادقة الثنائية (2FA) للمستخدم والمحاضرين | نظام حماية الرقم السري الاعتيادي بالموقع |
| بوابة شركاء ومواقع لمنح شهادات الجهات الشريكة | الرفع للشهادة ومراقبتها يتم بإدارة الموقع |
| دعم الوصول وفتح الموقع بدون نت (Offline PWA Mobile App) | واجهة موقع مرن ومتجاوب فقط (Responsive web App) |
